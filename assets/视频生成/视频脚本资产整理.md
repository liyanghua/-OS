# 视频脚本资产整理

> 来源：视频脚本生成界面截图  
> 用途：用于短视频脚本策划、分镜入库、拍摄表沉淀、视频资产结构化管理。

## 一、视频脚本概览（服饰展示类示例）

### 基本特征
- **脚本类型**：真人出镜商品展示 / 卖点短视频脚本
- **呈现形态**：表格化分镜脚本 + 拍摄指导 + 口播 / 台词 + 音效 / BGM
- **典型字段**：
  - 序号
  - 关键帧预览
  - 标题
  - 类别
  - 真人出镜
  - 景别
  - 运镜
  - 画面内容
  - 拍摄指导
  - 画面文案 / 花字
  - 口播 / 台词
  - 音效 / BGM
  - 时长

### 截图中可识别的分镜主题（示例）
1. 男子背身进门展示穿搭  
2. 弯腰取喷雾瓶动作  
3. 喷发胶整理发型  
4. 展示手部配饰并比赛  
5. 指袖口强调颜色细节  
6. 揭示内里摇粒绒材质感  
7. 整体造型展示与转体  
8. 指向品牌信息并笑谈  
9. 拉链开合演示功能  
10. 报价 / 价格感强化  
11. 双子图展示版型  

### 这类视频脚本适合抽象成的结构
- `video_script_asset`
- `video_storyline`
- `video_shot_spec`
- `shooting_instruction`
- `onscreen_copy_spec`
- `voiceover_spec`
- `bgm_spec`

## 二、多用途毛巾卖点展示类视频脚本

### 脚本名称
- **多用途毛巾，可用于洗_卖点展示类_视频脚本**

### 整体脚本逻辑
从截图中可以抽象出一条典型的卖点视频链路：

1. **强痛点开场**
2. **对比反差**
3. **核心卖点响应**
4. **多场景复用验证**
5. **长效价值强化**
6. **行动号召**

### 核心文案摘要
- 擦脸擦半天还湿漉漉？
- 包头发滴水弄湿衣服？
- 买两块毛巾还占地方，换得勤、洗完变硬？
- 这款多用途毛巾，吸水性直接拉满——擦脸 3 秒吸干水分，裹头发 5 分钟带走多余湿气，吹风机都能少开一半；
- 触感像云朵一样软，脸蛋和发丝都零刺激；
- 一巾顶俩，洗脸 + 干发全搞定，省下一半预算，腾出一半收纳空间；
- 更厉害的是，它抗菌防螨、耐洗不发硬，洗 30 次还是蓬松柔软；
- 学生党、长发女生、租房族、出差党，谁用谁知道；
- 现在下单，把低效洗护彻底换掉。

## 四、阶段文案拆解

### 1. 强痛点开场
- 擦脸擦半天还湿漉漉？
- 包头发滴水弄湿衣服？
- 买两块毛巾还占地方，换得勤、洗完变硬？别忍了！

### 2. 对比反差
- 这款多用途毛巾，吸水性直接拉满——
- 擦脸 3 秒吸干水分，裹头发 5 分钟带走多余湿气，吹风机都能少开一半；
