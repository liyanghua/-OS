│ 第 2 层：对象交互层 Object-Centric Co-Creation Layer               │
│ “围绕当前对象持续修改，而不是围绕空白 prompt 聊天”                  │
│----------------------------------------------------------------------│
│ 交互对象：机会卡 / Brief / 模板 / 第 N 张图位 / 标题候选 / 正文段落 │
│ 交互方式：                                                          │
│ - 点机会卡改方向                                                    │
│ - 点模板切换策略                                                    │
│ - 点图位改图位                                                      │
│ - 点标题改标题                                                      │
│ - 点正文结构改正文结构                                              │
│ 关键体验：                                                          │
│ - 当前对象锚点始终存在                                              │
│ - 局部编辑，不整套重来                                              │
│ - 版本比较与回退                                                    │
│ - 推荐动作 chips / 快速改写                                         │
│ 输出：经过人机共创收敛后的“准执行方案”                              │

                                  ↓

│----------------------------------------------------------------------│

│ - SKU 图方向                                                        │
│ - 详情图模块                                                        │
│ - 短视频脚本 / 分镜                                                 │
│ 核心能力：                                                          │
│ - 多版本派生                                                        │
│ - A/B 变体生成                                                      │
│ - 导出给设计师 / 生图工具 / 运营发布                                │
│ - 结果回流到机会卡与模板评估                                        │
